# meroByapar---Business-Management-System
MeroByapar is a full-stack web application designed to help small businesses manage and promote their products/services online. The system allows users to register businesses, display products, and interact through a centralized digital platform.

Technology Stacks used:
Frontend: Vite.React + Tailwind
Backend: Node + Express
Database: MySQL
Api: AXIOS

Features:
User authentication
Business listing
Product management 

To run this:
1. Create new folder and open it in vs code
2. copy this repo link and go to vs code terminal
3. clone it

For frontend:
folder structure :frontend/meroByapar/src "npm run dev"
For Backend:
folder structure :backend "node server.js"
